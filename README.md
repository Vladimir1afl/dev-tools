dev-tools
========================

## Зависимости
- Node.js 10.9.0+

## Установка
```bash
$ npm i
```

## Команды
Режим разработки
```bash
$ gulp
```
Сборка
```bash
$ gulp build
```

## Просмотр верстки
> localhost:port/html/desktop/

## Структура проекта
> assets/**/*.{html, js, (post)css} - исходники

> public/**/*.{html, js(es5), css} - сборка

> public/fonts - шрифты

> public/images - изображения

> public/node_modules/* - сторонние библиотеки для frontend
